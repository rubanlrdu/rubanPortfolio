Cyan Vote

cyan vote.

This project presents a practical exploration of blockchain technology through the creation of a decentralized electronic voting system. Focusing on a manageable scope, the system provides a platform for single- choice voting using Solidity smart contracts on a local Ethereum network. The project emphasizes learning core blockchain concepts such as smart contract development, distributed data storage, and web integration. The resulting application showcases voter registration, vote casting, and transparent result display. The development process highlights the challenges and opportunities in applying blockchain to real-world applications.
